How it works

A stage strip sits just below the header. Click any stage to see its content. The active stage gets an accent underline.

For the current work stage, an iteration sub-nav appears — numbered pills showing each run and its outcome. Click to navigate.

Past stages (planning, breakdown) show their artifact directly — no iterations to navigate since each had only one run.

Try clicking Planning or Breakdown above to see past stage artifacts, then click Work → iter 1 to see a rejection.

Key tradeoff: Two rows of nav chrome above the tabs. Clear hierarchy but takes vertical space.
