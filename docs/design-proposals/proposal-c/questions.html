Should pagination use cursor-based or offset-based pagination?
Cursor-based pagination is more performant for large datasets and
handles real-time inserts/deletes gracefully. Offset-based is simpler
to implement and allows jumping to arbitrary pages.
1 Cursor-based -- better for real-time data, more scalable
2 Offset-based -- simpler, supports page jumping
