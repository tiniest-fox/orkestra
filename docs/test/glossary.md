# Glossary

**Worktree** — An isolated working directory in Git that allows multiple branches to be checked out simultaneously without conflicts.

**Orchestrator** — The central coordination component that manages task lifecycle, agent spawning, and state transitions.

**Artifact** — Named output content produced by a stage execution, stored on the task and available as input to subsequent stages.

**Iteration** — A single execution run of an agent or script within a stage, with rejections creating new iterations for refinement loops.
